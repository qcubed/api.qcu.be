The QUpdateUtility is used by the qcodo_updater.cli and qcodo_downloader.cli command line utilities.

It will use the QCubed Updater WebService at http://api.qcodo.com/ to perform updates to the installed QCubed system.

Uses the cli_config.inc constants to convert a manifest-based file path to an actual file path on this system.

GetActualFilePath($strManifestFilePath) 

Will return NULL if the file being converted is specified to an actual path of "none" by the user.

Will return -1 if the file being converted does not have a path prefix (this will happen if a new path prefix has been added since the last time qcodo_updater was called)

An example $strManifestFilePath is (for example) INCLUDES|prepend.inc.php
