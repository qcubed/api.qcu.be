Returns the suggested MIME type for an actual file.

GetMimeTypeForFile(string $strFilePath) : string
Static

Using file-based heuristics (data points in the ACTUAL file), it will utilize either the PECL FileInfo extension OR the Magic MIME extension (if either are available) to determine the MIME type. If all else fails, it will fall back to the basic GetMimeTypeForFilename() method.

Parameters

$strFilePath

string

the absolute file path of the ACTUAL file

Returns

string
