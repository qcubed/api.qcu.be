    SqlVariable(mixed $mixData, boolean $blnIncludeEquality, boolean $blnReverseEquality) : string

    If IncludeEquality is set (usually not), then include an equality operator. So for most data, it would just be "=". But, for example, if $mixData is NULL, then most RDBMS's require the use of "IS".

    Parameters

    $mixData

    mixed

    $blnIncludeEquality

    boolean

    whether or not to include an equality operator

    $blnReverseEquality

    boolean

    whether the included equality operator should be a "NOT EQUAL", e.g. "!="

    Returns

    stringthe properly formatted SQL variable
